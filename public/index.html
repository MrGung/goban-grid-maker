Making Gobans - creating lines and hoshi

What?

This page will allow you to create templates of gobans. You can set up some options from which this page will create the grid.

That grid can be used as a cheap goban when printed on paper. But it could also be used as a template for creating high quality prints on wood (e.g. by screen printing it).

How?

This page generates a graphical representatin of the settings you provided. The format of that graphic is SVG, a vector format. That means it can be scaled without loss of quality.

Be aware, though, that the resulting graphic should not be transformed in any way since it is generated to fit the standard dimensions for gobans. Namely:

  • Horizontally the lines are 21.944mm apart.
  • The vertical spacing (disregarding the line-thickness) is 23.166mm.
  • If you choose to show coordinates, the I is omitted.

Tech-talk

You'll need a plugin to view the resulting graphic in your browser. I recommend the plugin from Adobe. If you don't use InternetExplorer have a look at SVG Viewer Plugin Installation for short and detailed information.

If you don't want to bother to install the plugin you don't have to. You should be able to download the file directly to your computer (right click 'save as') and process it with other software (e.g. CoreDraw can handle SVGs). I recommend Inkscape for creating, editing and viewing SVGs.
